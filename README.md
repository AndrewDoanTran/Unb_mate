The Student Helper App is a simple native Android application designed to support studentsâ€”especially new students or those overwhelmed with multiple deadlines. The app provides clear, practical tools that help users stay organized and navigate their tasks more efficiently.

This project is built entirely using the Android SDK and focuses on delivering a clean, minimal interface with meaningful functionality.

**Main features**
1. Reminders Assignment Manager

A lightweight reminder tool that allows students to:

Add assignments or tasks

Keep track of upcoming deadlines

Manage workload in a simple, organized list

This feature is ideal for students handling multiple courses, helping them stay on top of their responsibilities.

2. 2-D Direction Helper

An interactive 2-D direction feature designed to help new students understand directions visually.

Responds to touch input

Updates direction positioning interactively

Provides an intuitive way to understand simple directional guidance

This is especially useful for first-year students learning to navigate a new environment.

**Target users**
New students unfamiliar with navigating campus or new environments

Students with heavy assignment loads who need a simple tool to manage deadlines

Anyone looking for a quick, easy-to-use productivity tool on Android

**Purpose**
The goal of this app is to create a supportive tool for students, combining simple reminder management with an interactive direction helper to reduce stress and improve productivity.
